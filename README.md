# ReactCat-cli

自搭自用的React脚手架（React, Redux, Webpack, ES6）<br>
带有自己写的一些组件（按钮， 侧边栏， 日历等）

## 使用到的部分关键模块
* react-redux
* redux-actions
* redux-devtools
* react-router
* whatwg-fetch
* webpack-dev-server

## 安装

```bash
npm install
```

##启动

### 开发环境

```bash
npm run dev
```

### 生产环境

```bash
npm run deploy
```
